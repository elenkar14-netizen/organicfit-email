# OrganicFit — Email Template

Personalized onboarding email template for OrganicFit fitness app.

## 🔗 Live Preview

👉 **👉 https://https://elenkar14-netizen.github.io/organicfit-email**

## 📁 Structure

```
organicfit-email/
├── index.html          ← email template
├── images/
│   ├── hero.png        ← hero section image (replace placeholder)
│   └── start-today.png ← "Start Today" section image (replace placeholder)
└── README.md
```

## 🖼 Adding Images

Replace the placeholder images in the `images/` folder with your own:

- **hero.png** — right side of the hero block (recommended: 640×440px or larger)
- **start-today.png** — left side of the "Start Today" block (recommended: 400×260px or larger)

## 🚀 Deploy on GitHub Pages

1. Create a new repository on GitHub (e.g. `organicfit-email`)
2. Upload `index.html`, `README.md`, and the `images/` folder
3. Go to **Settings → Pages → Source** → select `main` branch, root `/`
4. Your page will be live at `https://YOUR_USERNAME.github.io/organicfit-email/`
